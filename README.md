# Sky Dishes 🍽️

A beautiful mobile recipe app mockup built with React + Tailwind CSS.

![Sky Dishes Preview](https://images.unsplash.com/photo-1512621776951-a57141f2eefd?w=800&h=400&fit=crop)

## Features

- 📱 **Mobile-first design** - iPhone-style frame mockup
- 🎨 **Beautiful teal theme** - Premium dark UI with translucent cards
- 🔄 **Smooth transitions** - Seamless screen navigation
- ♿ **Accessible** - ARIA labels and keyboard navigation
- 📦 **Zero dependencies** - Built with vanilla React + Tailwind

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
