# 🍽️ Sky Dishes - Recipe App

A beautiful mobile recipe application with an elegant iPhone-style interface, built with pure HTML, CSS, and JavaScript.

![Sky Dishes Preview](https://images.unsplash.com/photo-1512621776951-a57141f2eefd?auto=format&fit=crop&q=80&w=800)

## ✨ Features

### 📱 Screens
- **Welcome Screen** - Introduction with recipe categories
- **Recipe Detail** - Full recipe information with author details
- **Recipe Grid** - Cooking information at a glance
- **Cooking Steps** - Interactive step-by-step cooking guide

### 🎯 Interactive Features

| Feature | Description |
|---------|-------------|
| 🔍 **Search** | Search recipes by name or author |
| 👤 **Profile** | View user profile with stats |
| ❤️ **Favorites** | Toggle favorite recipes with animation |
| 📋 **Categories** | Browse recipe categories |
| 👨‍🍳 **Cooking Mode** | Interactive step tracking with progress bar |
| 📤 **Share Menu** | Share, save, or report recipes |

### 🎨 Design Features
- iPhone-style frame with notch
- Teal gradient theme (#1a4a42)
- Geometric SVG ornaments
- Smooth screen transitions
- Toast notifications
- Glassmorphism card effects
- Responsive mobile layout

## 🚀 Quick Start

Simply open `index.html` in any modern browser:

```bash
# Using Python
python -m http.server 8000

# Or using Node.js
npx serve
```

Then visit `http://localhost:8000`

## 📖 Usage

### Navigation
- Click **Continue** to move between screens
- Use the **back arrow** to return to previous screen
- Use **Start Cooking** or **View Steps** to enter cooking mode

### Search
1. Click the **search icon** in the top nav
2. Type to filter recipes
3. Click a result to view details

### Favorite Recipes
- Click the **heart icon** to toggle favorite status
- Toast notification confirms the action

### Cooking Steps
1. Navigate to the Steps screen
2. Click each step to mark it complete
3. Watch the progress bar fill
4. Complete all steps for congratulations!
